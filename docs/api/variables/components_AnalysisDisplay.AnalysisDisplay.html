    議論の分析結果をカード形式で表示するコンポーネント。

    AI分析により生成された議論の分析情報を視覚的に整理して表示します。 表示内容：

    • 主要な論点とその説明
    • 参加者ごとの立場と根拠
    • 対立点と争点の詳細
    • 進行状況評価と今後の方向性
    • 簡易統計情報（メッセージ数、参加者数など）
